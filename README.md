# Crown Conquest

A fantasy-themed React + Vite dashboard project.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

The production files will be created in the `dist` folder.

## Deploy

This project can be deployed on Vercel, Netlify, or any static hosting platform.

### Vercel

- Framework preset: `Vite`
- Build command: `npm run build`
- Output directory: `dist`

### Netlify

- Build command: `npm run build`
- Publish directory: `dist`

### Static hosting

1. Run `npm run build`
2. Upload the contents of `dist`

## Tech stack

- React
- Vite
- React Router
- Framer Motion
- Tailwind CSS
